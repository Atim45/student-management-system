# Student Management System

A simple console-based student management system in Python. Stores each student's name and grade in a dictionary, with automatic save/load to a text file so data persists between runs.

## Features

- **Add student** — enter a name and grade; automatically saved to file
- **Find grade** — look up a student's grade by name
- **Show all students** — print the full student-grade dictionary
- **Calculate average** — computes average grade point (A=4, B=3, C=2, D=1) across all students
- **Persistent storage** — records are saved to `student_list.txt` after every addition, and reloaded automatically on startup
- **Input validation** — handles non-integer menu input gracefully

## How to Run

```bash
python studentManagment1.py
```
